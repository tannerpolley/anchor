# Changelog

## 1.0.0-beta.1 — 2026-06-08

### Initial beta release

**Core**
- GitHub issue browsing: open/closed filtering, detail view with description + comments
- Pull request browsing: open/closed/merged filtering, detail view with description + timeline commits + comments
- Branch listing with single-click checkout via IntelliJ's native Git infrastructure
- Repository auto-detection from git remote configuration

**Rendering**
- Chromium-based HTML rendering (JCEF) for issue/PR descriptions and comments
- GitHub Flavored Markdown rendering via GitHub's official `/markdown` API
- Timeline view for PR commits with author, timestamp, and link to GitHub
- Full IDE theme mirroring — all rendered HTML adapts to light/dark theme changes
- Bubble-style containers for descriptions and comments matching IDE card styling

**UI**
- Three-tab tool window: Issues, PRs, Branches
- Phase-based sync indicators: "Fetching issues…" → "Fetching pull requests…" → "Fetching branches…"
- Loading states for detail views with progress text ("Rendering description…", "Loading comments 3 of 12…")
- Filter chips for open/closed/all states
- Keyboard shortcuts: R (refresh), Esc (back from detail)

**Actions**
- Open file on GitHub (right-click in editor)
- Clone repository from GitHub
- Create issue with label chips and assignee field
- Add remote from GitHub
- Create repository on GitHub

**Infrastructure**
- Markdown-to-HTML render cache (LRU, 200 entries)
- API response cache for comments and commits (LRU, 100 entries)
- HTML sanitizer — strips only JS execution vectors, preserves all visual content
- OkHttp-based GitHub REST API v3 client with pagination
- Persistent settings via IntelliJ's service infrastructure

**Branding**
- Anchor icon — monochromatic anchor silhouette adapting to IDE theme
- Marketplace icon with vibrant blue background
