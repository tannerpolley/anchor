# Milestones

Milestone pages are roadmap index views for Anchor. They describe why a bucket exists, the matching GitHub milestone title, and which specs, plans, or issue mirrors belong to it.

Canonical specs, plans, and issue mirrors stay in these roots:

- `docs/superpowers/specs`
- `docs/superpowers/plans`
- `docs/superpowers/issues`

Do not create nested canonical artifact folders under a milestone.
